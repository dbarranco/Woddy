# WODie 🏋️

Your AI-powered CrossFit programming buddy—science-backed workouts, static PWA, zero runtime API calls.

## What Is WODie?

Get personalized CrossFit programs and daily workouts built on real sports science. Every workout includes full session structure, equipment list, and scientific rationale with real citations.

**Two modes:**
- **Program mode** — structured 2/3/4-week programs with progressive loading
- **WOD mode** — rotating pool of daily workouts with category filters

## Key Features

- 🧠 **Science-backed** — every workout cites real sports science sources
- 📱 **Installable app** — works offline on iOS and Android
- ⚡ **Lightning fast** — pre-generated content, zero loading delays
- 🏃 **Pre-configured timer** — no setup needed, just hit start
- 📊 **Progressive loading** — periodized programs that actually work
- 🎯 **Smart constraints** — respects recovery, prevents overuse injuries

## How It Works

Every session is exactly 60 minutes:

1. **Static warmup** (5 min) — stretching and mobility
2. **Active warmup** (5 min) — movement preparation
3. **Main work** (20–25 min) — strength or skill, CNS-fresh
4. **Metcon** (10–20 min) — AMRAP, EMOM, or For Time
5. **Cooldown** (5–10 min) — stretching and mobility

No guesswork. No fluff. Just proven training methods.
