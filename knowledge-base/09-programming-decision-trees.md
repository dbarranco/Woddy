## Overview

This module synthesizes periodization theory and block programming frameworks to provide decision trees for session design, athlete profiling, and progression planning. The Tactical Barbell methodology provides a coach's perspective on session type selection and strength block structure, grounded in the concurrent training principles documented in previous modules.

## Session Type Selection Decision Tree

Use this tree to choose **what kind of session to program** (strength-emphasis, metcon-emphasis, skill-emphasis, aerobic, or deload).

```
START: What is the primary training goal for this session?

├─ Is it PEAK STRENGTH (heavy singles, doubles, triples)?
│  └─ → STRENGTH-EMPHASIS SESSION
│     Structure: Warmup + Strength Block (25 min, heavy, low reps) + Light metcon (10 min, short, low volume)
│     Example: 5×3 back squat @ 80–85% + 7 min AMRAP (light)
│     Recovery: 48h before next high-CNS session
│     Reference: Summers & Foreman recommend strength blocks as the "main event" with metcon as finisher
│
├─ Is it WORK CAPACITY or INTENSITY (sustained medium-to-high effort)?
│  └─ → METCON-EMPHASIS SESSION
│     Structure: Warmup + Moderate strength block (15 min, moderate) + Intense metcon (12–15 min, medium-to-high)
│     Example: 3×5 power clean @ 70% + 12 min AMRAP complex
│     Recovery: 36h typical
│     Reference: Build work capacity and aerobic power; conditioning is the primary stimulus
│
├─ Is it SKILL DEVELOPMENT (gymnastics, Olympic technique, movement quality)?
│  └─ → SKILL-EMPHASIS SESSION
│     Structure: Warmup + Skill Block (25–30 min, technical, low fatigue) + Light accessory (5 min)
│     Example: 15 min pull-up progressions + 10 min accessory work
│     Recovery: 24h (low systemic demand)
│     Rationale: Skill sessions demand central nervous system focus without metabolic fatigue
│
├─ Is it AEROBIC BASE or ENERGY SYSTEM DEVELOPMENT?
│  └─ → AEROBIC/STANDALONE SESSION
│     Structure: Warmup + Long monostructural (20–40 min, low intensity) + Cooldown
│     Example: 30 min steady-state row at conversational pace
│     Recovery: 24h (can stack with strength same week)
│     Reference: Tactical Barbell emphasizes separate aerobic blocks to minimize interference with strength adaptations
│
└─ Is it DELOAD or RECOVERY-FOCUSED?
   └─ → RECOVERY SESSION
      Structure: Warmup + Light accessory (15–20 min, <50% intensity) + Mobility (10 min)
      Example: Light dumbbell work + stretching
      Recovery: 24h; pair with another light session if needed
      Reference: Mandatory deload weeks reduce injury risk and allow supercompensation
```

**Coach's Note (Tactical Barbell):** Summers & Foreman emphasize clarity in session intent. The session type determines the metcon intensity, not the reverse. A strength-emphasis session will never have a high-intensity metcon; a metcon-emphasis session places light loading on the strength block. This prevents diluted stimulus and protects adaptations.

## Metcon Format Decision Tree

Use this tree to choose **AMRAP vs EMOM vs For Time** for the metcon block:

```
START: What is the metcon goal?

├─ Is the goal to teach PACING DISCIPLINE under fatigue?
│  └─ → AMRAP
│     Athlete must self-regulate intensity; later rounds will be slower than early ones.
│     Duration: 7–15 min typical, up to 20 min max
│     Example: 10 min AMRAP 5 pull-ups + 5 burpees
│     Reference: AMRAP requires athlete maturity in pacing (intermediate+)
│
├─ Is the goal CONSISTENT, CONTROLLED OUTPUT over time?
│  └─ → EMOM (Every Minute/2 Minutes/3 Minutes On the Minute)
│     Clock enforces pacing; athlete completes task within the minute, then rests.
│     Duration: 5–15 min (varies by interval and movement selection)
│     Example: E2MOM × 10 rounds: 8 power cleans + 8 ring dips
│     Reference: EMOM enforces rest-pause structure; ideal for technical complex movements or intermediate athletes
│
├─ Is the goal MAX EFFORT with FIXED WORK (racing)?
│  └─ → FOR TIME
│     All-out effort; no clock-driven pacing.
│     Duration: 3–10 min typical (rarely exceeds 12 min)
│     Example: 21–15–9 deadlifts + pull-ups for time
│     Reference: For Time demands cardiovascular tolerance and mental toughness
│
└─ Is the goal SUSTAINED, MODERATE EFFORT (aerobic or longer duration)?
   └─ → AMRAP (long duration) or EMOM (E3MOM+, extended intervals)
      Low intensity, pacing discipline more important than max speed.
      Duration: 15–20 min typical
      Example: 20 min AMRAP 200m row + 10 cal bike
      Reference: Long AMRAP or extended EMOM shifts toward aerobic system; intensity must be controlled
```

**Coach's Note (Tactical Barbell):** Format selection must align with the session intent. A strength-emphasis day uses only AMRAP or For Time for light finishers (not EMOM, which can interfere with recovery from heavy work). A metcon-emphasis day uses EMOM or For Time to maintain intensity. This clarity prevents the common mistake of mismatched stimulus.

## Strength Block Format Selection

Use this tree to choose **which strength progression scheme** to use in the strength block:

```
START: What week of the program are we in, and what is the goal?

├─ WEEK 1 (Baseline Establishment)
│  ├─ Is the goal technique refinement or new movement introduction?
│  │  └─ → RAMP BUILDUP (sets of 5→4→3→2 with increasing %)
│  │     Reason: Neurological preparation; athlete learns technique under increasing load
│  │     Example: Snatch warm-up into 5@65%, 4@72%, 3@78%, 2@83%
│  │     Citation: Ramp provides progressive CNS activation for complex movements
│  │
│  └─ Is the goal to establish baseline strength and volume?
│     └─ → STRAIGHT SETS (all sets same load)
│        Reason: Simplicity; establishes repeatable baseline
│        Example: 5×5 @ 72% back squat
│        Reference: Week 1 establishes the load for progression in weeks 2–3
│
├─ WEEK 2 (Moderate Intensity, Introduce Variety)
│  ├─ Is the athlete intermediate+ and should we exploit PAP (post-activation potentiation)?
│  │  └─ → WAVE LOADING (oscillating loads: 75%, 80%, 85%, repeat)
│  │     Reason: Post-Activation Potentiation; higher 3rd wave reps feel lighter due to neural priming
│  │     Example: 3@75%, 3@80%, 3@85%, repeat for 2–3 waves
│  │     Reference: Robbins 2005 - Post-Activation Potentiation; Summers & Foreman recommend waves for intermediate athletes
│  │
│  └─ Is this a moderate-intensity session or movement still in technique phase?
│     └─ → STRAIGHT SETS or RAMP
│        Example: 4×5 @ 78% (straight sets) or ramp if technique focus continues
│        Reference: Build intensity gradually; not all week 2 sessions require advanced schemes
│
├─ WEEK 3 (Peak Intensity)
│  ├─ Is the goal maximum strength with time efficiency?
│  │  └─ → TOP SET + BACKOFF
│  │     Reason: Hits peak intensity (1–3 reps heavy) + volume (sets of 5 lighter)
│  │     Example: 1×2@87%, then 4×5@70%
│  │     Reference: Bompa & Haff - Periodization; Summers & Foreman (Tactical Barbell 1) recommend this for concurrent training
│  │
│  └─ Is the goal sustained strength development (no drop-off in later sets)?
│     └─ → WAVE LOADING
│        Example: 3@82%, 3@85%, 3@88% (or higher if available)
│        Reference: Alternative to top set + backoff for athletes who tolerate higher volume
│
└─ WEEK 4 (Deload)
   └─ → STRAIGHT SETS at reduced intensity (60–65% 1RM)
      Example: 3×5 @ 65%
      Reason: CNS and tissue recovery; maintain movement quality without stimulus
      Reference: Mandatory deload prevents overtraining and allows supercompensation
```

**CRITICAL RULE:** Do NOT use wave loading or top set + backoff for novice athletes (Week 1 athletes or those with <3 months on a movement). Straight sets or ramp buildup only. Complex schemes demand neural readiness.

**Coach's Note (Tactical Barbell):** Summers & Foreman structure strength blocks around the block periodization principle: Week 1 establishes, Week 2–3 build, Week 4 recovers. Within this, the scheme (straight sets, ramp, wave, top set + backoff) must match the athlete's experience and the session intent. A strength-emphasis week uses heavier loads and lower reps; a metcon-emphasis week uses moderate loads and higher reps.

## Athlete Profile Decision Tree

Use this tree to adapt session prescription to athlete level and training age:

```
START: How advanced is the athlete?

├─ BEGINNER (0–6 months training, or new to CrossFit)
│  ├─ Strength focus: Ramp buildup + straight sets only
│  ├─ Loading: 60–70% 1RM if known; otherwise RPE 6–7
│  ├─ Metcon formats: AMRAP only (simplest pacing model)
│  ├─ Gymnastics cap: Strict pull-ups, step push-ups, elevated planks (no kipping, no muscle-ups)
│  ├─ Olympic lifting: NO barbell O-lifts in metcons; light dumbbell/kettlebell power work only
│  ├─ Session structure: Warmup + Short strength block (15 min, simple movements) + Light metcon (10 min) + Cooldown
│  ├─ Weekly frequency: 3–4 days (not 5 yet; foundational fitness still building)
│  └─ Recovery: Standard 24–36h between sessions
│     Rationale: Low complexity demands allow focus on movement quality and consistency
│
├─ INTERMEDIATE (6–18 months training)
│  ├─ Strength focus: Ramp buildup + straight sets OR light wave loading (if movement is familiar)
│  ├─ Loading: 70–80% 1RM; RPE 7–8
│  ├─ Metcon formats: AMRAP, EMOM (E2MOM or E3MOM), simple For Time
│  ├─ Gymnastics: Strict pull-ups, kipping pull-ups (with demonstrated prerequisite), ring dips, basic handstand work
│  ├─ Olympic lifting: Light power O-lifts in metcons (power clean, power snatch @ 60–70% 1RM); full O-lifts for strength block work only
│  ├─ Session structure: Standard 60 min (warmup + 20–25 min strength + 12–15 min metcon + 5 min cooldown)
│  ├─ Weekly frequency: 5 days (standard Woddy format)
│  └─ Recovery: Standard 24–48h; 48h after heavy lower body or intense metcon
│     Reference: Summers & Foreman (Tactical Barbell) place intermediate athletes at the "ready for block periodization" stage
│
└─ ADVANCED (18+ months, consistent training)
   ├─ Strength focus: Full range of schemes (straight sets, ramp, wave, top set backoff)
   ├─ Loading: 75–90% 1RM; RPE 8–9
   ├─ Metcon formats: All (AMRAP, EMOM, For Time, complex combinations)
   ├─ Gymnastics: Chest-to-bar pull-ups, muscle-ups, handstand push-ups, advanced static holds
   ├─ Olympic lifting: Full Olympic lifts in all metcon formats; technical complexity is not a limiting factor
   ├─ Session structure: Flexible (can handle 70+ min sessions, or high-intensity short sessions)
   ├─ Weekly frequency: 5–6 days (can sustain higher volume)
   └─ Recovery: Managed based on frequency and intensity; typically 24–48h between same stimulus
      Rationale: Advanced athletes tolerate varied stimuli and complex programming
```

**Coach's Note (Tactical Barbell):** Summers & Foreman emphasize the training age distinction: beginners need simplicity and consistency, intermediates are ready for block periodization and varied schemes, and advanced athletes can handle complex concurrent training. Mismatching athlete level to complexity (e.g., giving a beginner wave loading) reduces adaptation and increases injury risk.

## Metcon Intensity Prescription

Use this tree to set **intensity level** (low, moderate, high) for the metcon block based on session intent:

```
START: What intensity should this metcon be?

├─ STRENGTH-EMPHASIS SESSION (heavy lifting just happened)
│  └─ → LOW TO MODERATE METCON INTENSITY
│     Metcon is finisher, not primary stimulus. 10–12 min max, light load, moderate volume.
│     Example: 10 min AMRAP with light dumbbells, not barbell.
│     Intensity: 40–60% of max effort
│     Rationale: CNS is already taxed by heavy lifting; protect strength adaptation by keeping metcon light.
│     Reference: Wilson et al 2012 - Concurrent Training Meta-Analysis; Tactical Barbell emphasizes separation of stimulus
│
├─ METCON-EMPHASIS SESSION (Strength block was warm-up)
│  └─ → HIGH METCON INTENSITY
│     Metcon is the primary stimulus. 12–15 min, challenging load or high volume, competitive pace.
│     Example: 12 min AMRAP 8 power cleans + 12 ring dips (challenging but doable).
│     Intensity: 75–85% of max effort
│     Rationale: This session is designed to build work capacity and aerobic power.
│     Reference: Tactical Barbell 2 (Conditioning) emphasizes dedicated metcon days
│
├─ AEROBIC-EMPHASIS SESSION (no heavy lifting)
│  └─ → LOW TO MODERATE INTENSITY
│     Sustained effort, conversational pace, 20+ minutes.
│     Example: 30 min row at 60–70% max HR.
│     Intensity: 50–70% of max effort (polarized model)
│     Rationale: Aerobic base building requires lower intensity to avoid systemic fatigue and interference with strength
│     Reference: Summers & Foreman (TB2) recommend separate aerobic blocks; Gastin 2001 - Energy System Interaction
│
└─ SKILL-EMPHASIS SESSION (technique focus)
   └─ → NO METCON or VERY LIGHT
      Focus is entirely on skill refinement, not conditioning.
      If there's a finisher, keep it sub-5 min and light (e.g., "3 min easy rowing").
      Intensity: <50% of max effort if present
      Rationale: Protect CNS for skill development; fatigue impairs motor learning
```

**Coach's Note (Tactical Barbell):** Session intent determines metcon intensity. Never run a high-intensity metcon after heavy strength work; this creates uncontrolled stimulus variability and blunts both strength and endurance adaptations. Summers & Foreman advocate clarity: each session has one primary goal, and secondary work supports that goal without competing for recovery resources.

## Loading & Progression Guidelines by Goal

Load selection must match both the training week and the session goal. The following table applies to known 1RM values; use RPE (Rate of Perceived Exertion, 6–10 scale) if 1RM is unknown.

| Goal | Week 1 | Week 2 | Week 3 | Week 4 |
|---|---|---|---|---|
| **Hypertrophy** | 70–75% 1RM, 5×5 | 72–78% 1RM, 4×6 | 75–80% 1RM, 4×5 | 60–65% 1RM, 3×5 (deload) |
| **Strength** | 75–80% 1RM, 5×3 | 78–82% 1RM, 4×3 | 82–87% 1RM, 1×2 + backoff or wave | 65–70% 1RM, 3×5 (deload) |
| **Power** | 65–70% 1RM, 3×3 (explosive) | 70–75% 1RM, 3×3 (explosive) | 75–80% 1RM, 3×2 (explosive) | 60–65% 1RM, 3×3 (deload) |

**Progression Rule:** Progress load week-to-week only if the athlete completed all reps with clean form and within target rest periods last week. If any set failed or form degraded, hold load or reduce slightly. Do not force progress.

**Reference:** Bompa & Haff - Periodization 5th ed; Zatsiorsky & Kraemer - Science and Practice of Strength Training.

**Coach's Note (Tactical Barbell):** Summers & Foreman emphasize that load progression must align with the block's primary goal. A strength block (80%+ 1RM, lower reps) uses different progression curves than a hypertrophy block (70–75% 1RM, higher reps). Mixing goals within a single block reduces adaptations.

## Recovery & Sequencing

| Session Type | Typical Duration | Recovery Until Next Session | Can Stack Same Day? |
|---|---|---|---|
| Heavy strength (>80% 1RM) | 60 min | 48h | No (if next session also heavy) |
| Metcon-emphasis (high intensity) | 60 min | 36–48h | No (same day only if aerobic/skill) |
| Skill-emphasis (technical) | 30 min | 24h | Yes (pair with aerobic) |
| Aerobic-emphasis (low-moderate) | 30–40 min | 24h | Yes (pair with skill or light strength) |
| Deload (recovery-focused) | 45 min | 24h | Yes (very light; can pair with aerobic) |

**Sequencing Rules (Hard Constraints):**

1. **Never back-to-back high-CNS sessions** — heavy strength + intense metcon cannot be consecutive days
2. **Aerobic + skill can stack** — same day or consecutive days (low CNS demand)
3. **Deload weeks** — reduce intensity and volume across all sessions by 20–30%
4. **Heavy lower body followed by** — minimum 48h before next heavy lower body or intense metcon
5. **Olympic lifting** — max 3 sessions per week, never on consecutive days (recover CNS between sessions)

**Tactical Barbell Sequencing Philosophy:**

Summers & Foreman recommend a **block periodization approach** with clear weekly structure:

- **Day 1 (Monday):** Heavy strength emphasis (e.g., heavy squat + light finisher)
- **Day 2 (Tuesday):** Skill or aerobic (lower CNS demand; allows recovery from Day 1)
- **Day 3 (Wednesday):** Metcon emphasis (different stimulus from Day 1; non-overlapping muscle groups if possible)
- **Day 4 (Thursday):** Light or aerobic (active recovery)
- **Day 5 (Friday):** Strength or metcon (varies based on program goal)
- **Days 6–7 (Weekend):** Rest or very light movement

This pattern ensures recovery between high-demand sessions and prevents interference between strength and conditioning adaptations.

**Reference:** Summers & Foreman - Tactical Barbell 1 & 2; Bompa & Haff - Periodization 5th ed.

## Program Architecture

### Microcycle (Weekly Cycle)

A 5-day week is the standard Woddy format. Sessions are ordered to respect recovery demands:

1. **Heavy Strength + Light Finisher** (CNS-intensive)
2. **Skill or Aerobic** (CNS recovery, different stimulus)
3. **Metcon-Emphasis** (conditioning focus, different muscle groups if possible)
4. **Light Movement** (active recovery or skills)
5. **Strength or Metcon** (block-dependent)

### Mesocycle (4-Week Block)

Each 4-week block follows this structure:

- **Week 1:** Baseline establishment (60–75% 1RM, higher reps, straight sets or ramp)
- **Week 2:** Moderate intensity buildup (72–80% 1RM, varied schemes, introduce wave loading)
- **Week 3:** Peak intensity (80–87% 1RM, low reps, wave loading or top set + backoff)
- **Week 4:** Mandatory deload (60–65% 1RM, lower volume, straight sets only)

**Reference:** Bompa & Haff - Periodization 5th ed; Summers & Foreman - Tactical Barbell 1: Strength.

### Program Progression (Multiple Mesocycles)

- **2-week program:** Week 1 baseline + Week 2 peak (no deload, suitable for short blocks or athletes returning from layoff)
- **3-week program:** Week 1 baseline + Week 2 moderate + Week 3 peak (compressed; no deload)
- **4-week program:** Week 1–3 above + Week 4 mandatory deload (full periodization, suitable for sustained progress)

## Concurrent Training Interference Avoidance

Since CrossFit uses concurrent training (strength and conditioning in the same week), programming must minimize the **interference effect** — endurance training can blunt strength and hypertrophy adaptations.

**Tactical Barbell Solution (Summers & Foreman):**

1. **Separate stimuli by emphasis** — each session has one primary goal
2. **Order blocks within the session** — strength work first (CNS fresh), light conditioning after
3. **Separate by time** — heavy strength days are followed by low-CNS days (skill, aerobic) before returning to high-demand conditioning
4. **Manage volume** — do not combine high-volume strength + high-volume conditioning on the same day

**Reference:** Wilson et al 2012 - Concurrent Training Meta-Analysis; Tactical Barbell 2: Conditioning.

## Key Decision Summary

When designing a session or week, ask in order:

1. **What is the PRIMARY goal?** (strength, conditioning, skill, aerobic, recovery)
2. **What is the athlete's level?** (beginner, intermediate, advanced)
3. **What week of the block are we in?** (1, 2, 3, or 4)
4. **What secondary goal supports the primary?** (e.g., if primary is strength, secondary is light conditioning)
5. **Is there adequate recovery before/after this session?** (check sequencing rules)

Answer these five questions and the session design follows logically. Ambiguity in any of these leads to unclear stimulus and poor adaptation.
